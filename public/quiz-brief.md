ข้อ 1 : คุณยืนอยู่หน้าประตูรากไม้ท่ามกลางฟ้าสว่างที่ดูเหงาหงอยและคิวที่ยาวเหยียด

1. ฉันควรได้สิทธิพิเศษเข้าก่อนใคร -> A+1
2. กังวลว่าคนจะมองฉันไม่ดีหรือเปล่า -> B+1
3. แบ่งน้ำให้แฟรี่ข้างๆ ที่ดูเหนื่อยล้า -> C+1
4. สังเกตการณ์หาคนโกงคิวอย่างเงียบๆ -> D+1

ข้อ 2 : คุณพบแฟรี่เด็กปีกฉีกขาดซ่อนตัวอยู่ในพุ่มไม้ ดวงตาคู่นั้นเต็มไปด้วยความสิ้นหวัง

1. มองว่าน่ารำคาญและบินผ่านไป -> A+1
2. คิดว่าโชคชะตาใจร้าย (เหมือนที่ทำกับฉัน) -> B+1
3. เข้าไปกอบกุมมือและร้องไห้ไปกับเขา -> C+1
4. รีบทำแผลให้และสอนวิธีระวังตัว -> D+1

ข้อ 3 : ในงานเลี้ยงที่แสงนวลตาคุณเห็นทุกคนหัวเราะ แต่ลึกๆคุณกลับรู้สึกถึงบางอย่าง

1. พยายามไปยืนในที่สว่างที่สุดเพื่อให้ทุกคนสนใจ -> A+1
2. ยืนที่มุมมืด แอบหวังว่าจะมีใครสักคนสังเกตเห็นฉัน -> B+1
3. เดินแจกรอยยิ้มและดูแลความรู้สึกคนที่ดูเหงาๆ -> C+1
4. ยืนคุมเชิงมองหาพฤติกรรมแปลกๆ ที่ซ่อนอยู่หลังงาน -> D+1

ข้อ 4 : เมื่อส่องกระจกโบราณจิตวิญญาณของคุณสั่นสะท้านกับภาพที่ปรากฏออกมา

1. เห็นภาพตัวเองเป็นราชาที่สง่างาม -> A+1
2. เห็นภาพตัวเองเป็นเหยื่อที่ถูกทำร้าย -> B+1
3. เห็นภาพดวงใจที่ส่องสว่างด้วยความรัก -> C+1
4. เห็นภาพนักรบที่มองทะลุทุกคำลวง -> D+1

ข้อ 5 : พายุเวทมนตร์พัดถล่มความกลัวเริ่มเกาะกินใจทุกคนจนเริ่มเห็นแก่ตัว

1. ผลักคนอื่นออกเพื่อให้ฉันเข้าที่บังแดดก่อน -> A+1
2. ร้องไห้ตัดพ้อว่าทำไมเรื่องซวยต้องเกิดกับฉัน -> B+1
3. ใช้ปีกตัวเองบังฝนให้เพื่อนจนตัวเองเปียกโชก -> C+1
4. จัดระเบียบการหลบภัยและกันพวกที่เห็นแก่ตัวออก -> D+1

ข้อ 6 : คุณถูกชี้หน้าว่าเป็นกาลกิณีเพื่อนที่เคยไว้ใจกลับมองคุณด้วยความระแวง

1. ตอกกลับด้วยความโกรธว่า "แกกล้าดียังไง!" -> A+1
2. เสียใจจนพูดไม่ออก และคิดว่าไม่มีใครรักฉันจริง -> B+1
3. ยอมรับความผิดไว้เองเพื่อให้ทุกคนสบายใจ -> C+1
4. สวนกลับด้วยความจริงและแฉแผนการของคนคนนั้น -> D+1

ข้อ 7 : การจะรอดออกไป คุณต้องสละ'แสงแห่งชีวิต' ซึ่งอาจทำให้คุณอ่อนแรงไปชั่วนิรันดร์

1. รอให้คนอื่นทำ "คนอย่างฉันสำคัญเกินกว่าจะเสียพลัง" -> A+1
2. "ถ้าฉันทำแล้ว ทุกคนจะเห็นค่าความลำบากของฉันไหม?" -> B+1
3. ยินดีสละพลังทั้งหมดที่มีเพื่อให้ทุกคนปลอดภัย -> C+1
4. ยอมสละพลังส่วนหนึ่ง และบังคับให้คนที่เอาเปรียบช่วยด้วย -> D+1

ข้อ 8 : เมื่อรอดมาได้ ทุกคนต่างซาบซึ้งและเข้ามาโอบกอดคุณ

1. "แน่นอน ถ้าไม่มีฉัน พวกคุณก็ไม่รอดหรอก" -> A+1
2. "ไม่เป็นไร (แต่ในใจรู้สึกว่าคำขอบคุณมันน้อยไป)" -> B+1
3. "แค่เห็นทุกคนยิ้มได้ ฉันก็มีความสุขที่สุดแล้ว" -> C+1
4. "ยินดีที่ช่วย แต่คราวหลังจงดูแลตัวเองให้ดีกว่านี้" -> D+1

ข้อ 9 : ก่อนจากลา คุณจารึกข้อความด้วยหยดน้ำตาและพลังใจลงบนแผ่นหิน

1. "จงก้มหัวให้ผู้ที่เหนือกว่า" -> A+1
2. "โลกนี้ช่างใจร้ายกับฉันเหลือเกิน" -> B+1
3. "ความรักคือเวทมนตร์ที่ยิ่งใหญ่ที่สุด" -> C+1
4. "จงมีใจเมตตา แต่ต้องมีตาที่มองเห็นความจริง" -> D+1
